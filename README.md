🐰게임 스토리🐰


🐾전세사기를 당한 미미가 도시락 가게를 열어 손님들에게 도시락을 팔아요!🐾

🐾미미는 2층 침대에서 일어난 후, 1층으로 내려가요, 1층에 오른쪽 하단에 있는 주방이랍니다!🐾

🐾그러면 손님들이 와서 도시락을 주문하고 싶다고 말해요🐾

🐾미미는 그렇게 도시락을 만드는걸로 하루를 시작해요 🐾


🐰게임 설명🐰

1. 주문서를 보고 재료를 도시락에 넣자 -
2. 1분안에 완성해야해요! 아니면 도시락을 만들 수가 없어요!! (´⊙‸⊙`)
3. 완성하면 손님들에게 갖다줘요 -
4. 손님들의 주문을 다 끝냈으면 하루는 끝! ๛(ｰωｰ˘)
5. 미미는 다시 자신의 방으로 돌아가요🐾🐾🐾
